Introduction

Web programming is the process of developing websites using technologies such as HTML, CSS and JavaScript. The course SCSV 1223 – Web Programming introduces the fundamental concepts required to design, develop and deploy a functional website.

What I Have Learned

HTML

Through this course, I learned how to structure web pages using HTML elements such as headings, paragraphs, lists and semantic tags. HTML provides the basic framework of a website.

CSS

CSS is used to control the appearance and layout of a website. I learned how to apply styles such as colors, fonts and spacing to improve readability and overall user experience.

JavaScript

Basic JavaScript was introduced to add simple interactivity to a website. Examples include button click events and alert messages.

Learning Reflection

Throughout this course, I gained a better understanding of how HTML, CSS and JavaScript work together to create a complete website. Initially, I faced challenges in understanding the structure and organization of files.

However, with continuous practice and guidance, I became more confident in developing web pages and deploying them online using GitHub Pages. This learning experience has enhanced my problem-solving skills and increased my interest in web development.
