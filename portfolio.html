Introduction to Web Programming

Web programming involves the development of websites using technologies such as HTML, CSS and JavaScript. This course introduces the fundamental concepts required to build a functional and well-structured website.

What I Have Learned

HTML

I learned how to structure a web page using HTML elements such as headings, paragraphs, lists and semantic tags.

CSS

CSS is used to control the layout, colors and overall appearance of the website to improve user experience.

JavaScript

Basic JavaScript is used to add interactivity such as button click events.

Learning Reflection

Through this course, I gained a better understanding of how HTML, CSS and JavaScript work together to create a complete website. Although I faced challenges in understanding the structure at first, continuous practice helped me improve my skills and confidence in web programming.
